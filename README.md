## Contributing
`fork the repository`

```
git clone https://github.com/{your github username here}/Bankings-system
```
```
cd Bankings-system
```

`create a .env file`

`write DB_URL="insert your mongodb uri here"`

```
npm install
```

```
npm start
```

`Project will start running at http://localhost:5000`
